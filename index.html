🚀 Learn Git Commands

Master the essential Git commands for version control

Setup & Configuration

Configure your Git identity and preferences:

git config --global user.name "Your Name"
git config --global user.email "your@email.com"
Example: Set your Git user information for commits

Basic Commands

Initialize a repository:

git init

Clone a repository:

git clone <repository-url>
Example: git clone https://github.com/user/repo.git

Staging & Committing

Check status:

git status

Stage files:

git add <filename> # Add specific file
git add . # Add all changes

Commit changes:

git commit -m "Commit message"
Example: git commit -m "Fix: resolved login bug"

Branching

Create and switch branches:

git branch <branch-name> # Create branch
git checkout <branch-name> # Switch branch
git checkout -b <branch-name> # Create and switch

List branches:

git branch # Local branches
git branch -a # All branches

Remote Operations

Push and pull changes:

git push origin <branch-name> # Push to remote
git pull origin <branch-name> # Pull from remote

View remotes:

git remote -v

Merging & Rebasing

Merge branches:

git merge <branch-name>

Rebase:

git rebase <branch-name>
Example: git merge develop (merge develop into current branch)

History & Logs

View commit history:

git log # View all commits
git log --oneline # Condensed view
git diff # Show unstaged changes

Undo Changes

Revert changes:

git restore <filename> # Discard changes
git restore --staged <filename> # Unstage file
git revert <commit-hash> # Create new commit that undoes changes
git reset --hard <commit-hash> # Reset to specific commit
